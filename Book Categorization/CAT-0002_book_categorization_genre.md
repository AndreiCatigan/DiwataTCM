# Test Case ID: CAT-0002
## Test Case Name: Book categorization - Genre

**Summary:** Verify that the system accurately displays books grouped according to literary or technical genres.

**Preconditions:**
1. The user has navigated to the library catalog sorting interface.

| # | Step Actions | Expected Results |
|---|---|---|
| 1 | Locate the genre selection controller on the user interface. | Verify that available genres (e.g., Fiction, Reference, Technical Manuals, Textbooks) are visible. |
| 2 | Select a specific genre option (e.g., "Reference") from the list. | Verify that the system dynamically filters the grid to show books matching the chosen genre. |
