# Test Case ID: CAT-0001
## Test Case Name: Book categorization - Subject

**Summary:** Verify that users can successfully filter and view available school library books organized by specific academic subjects.

**Preconditions:**
1. The user has navigated to the library catalog sorting interface.
2. Book records are correctly cataloged under subject taxonomies in the database.

| # | Step Actions | Expected Results |
|---|---|---|
| 1 | Locate the subject filtering panel or dropdown selector. | Verify that a comprehensive list of school subjects (e.g., Computer Science, Physics, Mathematics) displays correctly. |
| 2 | Select a specific subject (e.g., "Computer Science") and click the filter button. | Verify that the book view updates dynamically to present only the records cataloged under that specific subject. |
