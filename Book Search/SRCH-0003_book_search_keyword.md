# Test Case ID: SRCH-0003
## Test Case Name: Book search - Keyword

**Summary:** Verify that the system successfully displays relevant book results when a user queries a general textbook topic or technical keyword.

**Preconditions:**
1. The user has navigated to the library book search interface.

| # | Step Actions | Expected Results |
|---|---|---|
| 1 | Click on the main search input bar field. | Verify that the field is active and ready for text input. |
| 2 | Input a general technical keyword (e.g., "Database") into the input field and click the "Search" button. | Verify that the system fetches and displays all book records that contain the keyword in their title, description, or metadata tags. |
